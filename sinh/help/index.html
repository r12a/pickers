List of character apps

Updated 19 November, 2022 • recent changes pickers/sinh • leave a comment

This Unicode character app allows you to produce or analyse runs of Sinhala text using the Sinhala script. Character apps are especially useful for people who don't know a script well, as characters are displayed in ways that aid identification.

The app includes characters in the Unicode Sinhala block that are used for writing text in the Sinhala language.

If something is broken or missing raise an issue. For release notes see the Github commit list. The pictures of the UI used in this page are taken from a variety of character apps.
