List of character apps

Updated 3 December, 2021 • recent changes pickers/latn-ff • leave a comment

This Unicode character app allows you to produce or analyse runs of Fula text using the Latin script. Character apps are especially useful for people who don't know a script well, as characters are displayed in ways that aid identification.

The app includes characters in the Unicode Latin blocks that are used for writing text in the Fula language.

If something is broken or missing raise an issue. For release notes see the Github commit list. The pictures of the UI used in this page are taken from a variety of character apps.
